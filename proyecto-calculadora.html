Sobre el Proyecto

Este proyecto nació con el objetivo de consolidar las bases de la programación estructurada en Python. Más allá de realizar operaciones matemáticas básicas, el desafío fue crear un sistema a prueba de fallos que pudiera manejar entradas inesperadas del usuario sin colapsar.

Lógica Implementada

Se utilizaron estructuras de control modernas como match-case (Python 3.10+) para optimizar la selección de operaciones y mejorar la legibilidad del código.

def calcular(num1, num2, operacion):
    try:
        match operacion:
            case '+':
                return num1 + num2
            case '/':
                if num2 == 0:
                    return "Error: División por cero"
                return num1 / num2
            # ... más operaciones
    except ValueError:
        return "Entrada inválida"

Características Clave

  • Validación de Tipos: Asegura que solo se procesen números.
  • Manejo de Errores: Control de división por cero y desbordamientos.
  • Interfaz Limpia: Interacción clara con el usuario mediante consola/HTML.
